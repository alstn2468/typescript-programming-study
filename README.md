## 스터디 방법

1. 1주일 동안 스터디할 범위를 정합니다.
2. 각자 스터디를 한 후 마크다운 형식으로 정리합니다.
3. 매주 각자의 스터디 내용을 공유합니다.
4. 타입스크립트를 활용하여 사이드 프로젝트를 진행합니다.
